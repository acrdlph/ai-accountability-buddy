<decisions_made>

- Hardcoded unit="rep" in add_habit_log: LLM was passing invalid units like "hours"
- Switched briefing model from gpt-4o-mini to gpt-5-mini per user request
- OAuth fix: Habitify OIDC requires prompt=consent to grant offline_access scope
