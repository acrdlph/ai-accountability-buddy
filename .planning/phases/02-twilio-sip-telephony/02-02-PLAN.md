**1. Add `detected_answering_machine` method** to `AccountabilityAgent` class:
```python
@function_tool()
async def detected_answering_machine(self, ctx: RunContext):
    """Called when the call reaches voicemail. Use this tool AFTER you hear the voicemail greeting"""
    logger.info("Voicemail detected -- hanging up without leaving a message")
    job_ctx = get_job_context()
    await job_ctx.api.room.delete_room(
        api.DeleteRoomRequest(room=job_ctx.room.name)
    )
```
This method goes AFTER the `on_enter` method, BEFORE the class closing.

**2. Update SYSTEM_PROMPT** -- append this rule to the existing Rules section (before the closing `\"`):
```
- If you hear a voicemail greeting or automated system instead of a real person, \
immediately call the detected_answering_machine tool. Do not leave a message.
```
This instruction MUST be in the system prompt or the LLM will not know to call the tool.

**Key constraints:**
- The `@function_tool()` decorator makes the method auto-discovered by the agent framework -- it does NOT need to be added to the `tools=` list in `__init__`. The agent framework discovers `@function_tool` methods on Agent subclasses automatically.
- Do NOT remove EndCallTool from the tools list -- both tools coexist
- The voicemail tool uses `delete_room` (same mechanism as EndCallTool) to hang up
- Use `get_job_context()` inside the tool to access the room API (not `self.session`)
